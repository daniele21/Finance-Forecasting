# Time Series Analysis applied to Finance

## Models:

  - Linear NN
  - Conv-LSTM NN
  
## Prediction on:

  - UAL --> United Arlines
  - DAL --> Delta Airlines
  - AAL --> American Airlines
  - LHA --> Lufthansa
  
  
## Model Details:

  - Training loss: Huber loss
  - Evaluation: MAE
  - Optimizer: Adam
